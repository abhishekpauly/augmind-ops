# Core Terms

## Overview
This page defines foundational terms that appear frequently across AugMind Ops documentation.

## How It Works
Each entry provides a concise definition and practical context.

## Example
### API (Application Programming Interface)
A set of rules that allows one software system to communicate with another. APIs define how requests are made and what responses are returned.

### SDK (Software Development Kit)
A package of tools, libraries, and documentation that helps developers build applications for a specific platform or service.

### Webhook
A mechanism where one system automatically sends real-time data to another system when an event occurs.

### Vector Database
A database designed to store and search vector embeddings, commonly used for semantic search and retrieval in AI applications.

## Key Takeaways
- APIs enable system-to-system communication.
- SDKs speed up development on specific platforms.
- Webhooks support event-driven integration.
- Vector databases power similarity search for AI workflows.
